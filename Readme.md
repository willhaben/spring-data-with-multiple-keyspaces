# Spring Boot With Multiple Keyspaces

Sample repository for the related [willhaben tech blog post](https://tech.willhaben.at/). You can also read it as a Markdown file 
[here](./Blogpost.md).

## How To Run

### Prerequisites

* Java >= 17
* Maven
* Docker (for testcontainers)

Only a sample application only working within the test context. Run `mvn test` to run the tests. See the blogpost and code
for further details.
